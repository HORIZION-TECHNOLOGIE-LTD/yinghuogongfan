# Design References and Inspiration

This document tracks design inspiration and references for the SurfSense landing page and UI components.

## Landing Page Inspirations

### GrapesJS Website
- **Repository**: https://github.com/GrapesJS/website.git
- **Live Site**: https://grapesjs.com/
- **Key Features**:
  - Modern AI-powered landing page interface
  - Clean, minimalist design with strong visual hierarchy
  - Dark theme with excellent contrast
  - Rotating headline text for engagement
  - Interactive prompt interface in hero section
  - Smooth animations and transitions
  - Mobile-responsive design
  - Clear call-to-action buttons

**Notable Design Patterns**:
- Sticky header with backdrop blur effect
- Large, bold hero headlines with rotating text
- Template gallery showcasing capabilities
- Clean footer with organized link sections
- GitHub integration and star count display

### Design Principles Applied to SurfSense

Our landing page follows modern SaaS design principles:
- **Visual Hierarchy**: Clear progression from hero to features to CTA
- **Animation**: Subtle motion design with beam effects and smooth transitions
- **Accessibility**: Dark/light mode support, semantic HTML
- **Performance**: Next.js 15 with optimized images and code splitting
- **Responsiveness**: Mobile-first design approach
